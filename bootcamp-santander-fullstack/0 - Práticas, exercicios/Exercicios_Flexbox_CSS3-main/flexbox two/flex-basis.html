flex-basis definição:

propriedade que estabelece o tamanho inicial do item antes das distribuição de espaço restante dentro dele, usando como base o conteúdo interno disposto.

Valores possíveis:

  • auto:caso o item não tenha tamanho,este será proporcional ao conteúdo do item
  • px, %, em,...: são valores exatos previamente definidos
  • 0(zero): terá relação com a definição do flex-grow
